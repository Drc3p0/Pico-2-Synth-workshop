# Pico Music Workshop — Config Generator

A browser-based tool that generates `code.py` files for the Pico Music Workshop.

## Usage

1. Open `index.html` in any web browser (double-click or drag into browser)
2. Choose your track: **Synth** or **MIDI**
3. Adjust the options to your liking
4. Click **Copy to Clipboard** (or **Download**) and paste into your Pico's `code.py`

## Requirements

- Any modern browser (Chrome, Firefox, Safari, Edge)
- No internet connection needed — works fully offline
- No build step — just open the file
